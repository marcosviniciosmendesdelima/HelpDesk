# Helpdesk 

Sistema de gerenciamento de chamados (Help Desk) desenvolvido para organizar e controlar solicitações de suporte técnico.

## Descrição

Este projeto tem como objetivo criar uma aplicação para gerenciamento de tickets de suporte, permitindo que usuários registrem problemas e que a equipe técnica acompanhe e resolva essas solicitações.

O sistema facilita a organização do suporte técnico dentro de empresas ou equipes de TI.


## Objetivo do Projeto

O objetivo principal é aplicar conceitos de desenvolvimento de software como:

- Organização de código
- Arquitetura de software
- Separação de responsabilidades
- Boas práticas de desenvolvimento

Além disso, o projeto serve como prática para desenvolvimento de sistemas de suporte e gerenciamento de chamados.
